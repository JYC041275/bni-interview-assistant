# 🚀 快速開始 - 5 分鐘部署到 GitHub Pages

## 📝 超簡化版本（跟著做就對了！）

### 第一步：檢查 Git 是否已安裝

打開「終端機」(Terminal)，複製貼上這行：
```bash
git --version
```

**看到版本號？** ✅ 太好了，繼續下一步！  
**看到錯誤？** ❌ 先安裝 Git（看下面的說明）

#### 如何安裝 Git（Mac 用戶）
```bash
xcode-select --install
```
點擊「安裝」，等待完成即可。

---

### 第二步：設置 Git（只需要做一次）

複製貼上這兩行（記得改成您的資訊）：
```bash
git config --global user.name "您的名字"
git config --global user.email "您的郵箱@example.com"
```

---

### 第三步：準備專案

```bash
# 進入專案資料夾
cd /Users/jryao/工作用資料庫/智能工作流/BNI會員訪談會議記錄

# 安裝所需套件
npm install
```

---

### 第四步：在 GitHub 創建倉庫

1. **登入 GitHub**：https://github.com
   - 還沒帳號？點右上角 "Sign up" 註冊（免費）

2. **創建新倉庫**：
   - 點右上角 "+" → "New repository"
   - Repository name：`bni-interview-assistant`
   - 選擇 **Public**（這樣才能免費使用 GitHub Pages）
   - **不要勾選** "Add a README file"
   - 點擊 "Create repository"

3. **複製倉庫網址**：
   - 在新頁面上，找到類似這樣的網址：
   ```
   https://github.com/您的用戶名/bni-interview-assistant.git
   ```
   - 複製這個網址（等下會用到）

---

### 第五步：上傳代碼到 GitHub

回到終端機，執行以下命令：

```bash
# 初始化 Git
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit"

# 連接到 GitHub（請替換成您剛才複製的網址）
git remote add origin https://github.com/您的用戶名/bni-interview-assistant.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

> 💡 **提示**：第一次推送時，可能會跳出登入視窗，輸入您的 GitHub 帳號密碼即可

---

### 第六步：部署到 GitHub Pages（最後一步！）

只需要一行命令：

```bash
npm run deploy
```

**等待 1-2 分鐘**，完成後會顯示：
```
Published
```

---

### 🎉 完成！查看您的網站

您的網站網址是：
```
https://您的用戶名.github.io/bni-interview-assistant/
```

**例如**：如果您的 GitHub 用戶名是 `jryao`，網址就是：
```
https://jryao.github.io/bni-interview-assistant/
```

---

## 🔄 之後如何更新網站？

當您修改了代碼，想要更新網站時：

```bash
# 1. 提交更改
git add .
git commit -m "更新說明"
git push

# 2. 重新部署
npm run deploy
```

就這麼簡單！

## ⚠️ 可能遇到的問題

### 問題 1：`git: command not found`
**解決方法**：安裝 Git
```bash
xcode-select --install
```

### 問題 2：推送時要求密碼，但密碼不對
**解決方法**：GitHub 現在需要使用 Personal Access Token

1. 前往：https://github.com/settings/tokens
2. 點擊 "Generate new token" → "Generate new token (classic)"
3. 勾選 `repo` 權限
4. 點擊 "Generate token"
5. **複製這個 token**（只會顯示一次！）
6. 在終端機要求輸入密碼時，貼上這個 token

### 問題 3：網站顯示 404
**解決方法**：
1. 前往 GitHub 倉庫頁面
2. 點擊 "Settings" → "Pages"
3. 確認 Source 設置為 `gh-pages` 分支
4. 等待 1-2 分鐘

### 問題 4：`npm: command not found`
**解決方法**：安裝 Node.js
1. 前往：https://nodejs.org/
2. 下載並安裝 LTS 版本
3. 重新打開終端機
